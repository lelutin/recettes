# Ceci est un site web

Recueil de mes recettes. Vous pouvez voir les recettes mises en page sur
<https://recettes.lelutin.ca/>

Ce site utilise [hugo](https://github.com/gohugoio/hugo) avec le thème
[quint](https://github.com/victoriadrake/hugo-theme-quint) pour le formatage.

## Accès direct aux fichiers

Toutes les recettes sont en format Markdown. Si vous voulez accéder aux fichiers
markdown direct sans passer par le site web, vous trouverez tous les fichiers
sous [content/recettes/](content/recettes/) et aussi
[content/techniques_de_base](content/techniques_de_base).
